# Coffee-Shop-Sales-Analysis-Dashboard
📌 Overview

This project focuses on analyzing Coffee Shop sales data using Microsoft Excel to uncover valuable business insights. An interactive dashboard was created using Pivot Tables, Pivot Charts, Slicers, and Excel formulas to monitor sales performance, customer footfall, product demand, and purchasing trends.

The dashboard enables stakeholders to make data-driven decisions by visualizing key performance indicators (KPIs) and identifying sales patterns across different store locations, product categories, weekdays, and business hours. The project aims to improve business performance through meaningful data analysis.

🚀 Project Objectives
Analyze total sales and customer footfall
Identify peak business hours
Compare sales across store locations
Analyze product category performance
Identify top-selling products
Understand customer ordering behavior
Compare order distribution by drink size
Monitor weekday sales trends
Build an interactive dashboard using Excel

📊 Key Insights
Coffee is the highest revenue-generating product category.
Morning hours receive the highest customer orders.
Hell's Kitchen store records the highest overall sales.
Barista Espresso is the top-selling product.
Regular-sized drinks account for the largest share of total orders.
Weekday sales remain relatively consistent with slight variations.
